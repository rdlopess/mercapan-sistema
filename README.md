# Mercapan Sistema

Sistema web de gestão para distribuidora.

## Stack
- **Frontend**: React + Vite + Tailwind CSS
- **Backend**: Node.js + Express
- **Banco**: Supabase (PostgreSQL)
- **Deploy**: Railway

## Módulos
- Produtos e categorias
- Fornecedores e cotações
- Tabela semanal de preços
- Clientes e pedidos

## Setup

### Backend
```bash
cd backend
cp .env.example .env
# Preencha as variáveis no .env
npm install
npm run dev
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
